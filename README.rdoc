== README

Для локального запуска всё стандартно.
База используется mongoDB.

Деплой на продакшен-сервер:
На сервере используется nginx для статики.
thin как основной веб сервер.
Команда для запуска
thin start -C /opt/HS-garland/config/thin.yml
Под рутом не установлены рельсы.
Их надо запускать под пользователем corehook

JS и CSS минимизированые лежат в public
